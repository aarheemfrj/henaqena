# قائمة الإطلاق

## قبل TestFlight وGoogle Play

- [ ] إدخال مفاتيح WhatsApp/SMS والبريد في بيئة الإنتاج.
- [ ] تفعيل Firebase وAPNs.
- [ ] اختيار مزود الخرائط وإضافة مفاتيح iOS/Android.
- [ ] ضبط `ADMIN_API_KEY` و`ADMIN_SESSION_SECRET` بقيم عشوائية قوية.
- [ ] تشغيل HTTPS وReverse Proxy.
- [ ] اختبار `/health` و`/ready` من خارج الخادم.
- [ ] تفعيل النسخ الاحتياطي واختبار الاستعادة.
- [ ] نقل الصور إلى Object Storage قبل التوسع.
- [ ] مراجعة سياسة الخصوصية وشروط الاستخدام وحذف الحساب.
- [ ] تجربة مغلقة على 20–50 مستخدمًا.
- [ ] تجهيز Bundle ID وPackage Name وStore listings.
- [ ] اختبار التسجيل، الدخول، البحث، إضافة النشاط، الإعلان، التقييم والإشعارات على هاتف حقيقي.
