# Hena Qena brand assets

ضع هنا ملفات الهوية النهائية بعد اعتمادها:

- `logo.svg` — الشعار الأساسي المتجهي.
- `logo-mark.svg` — الأيقونة وحدها.
- `logo-horizontal.svg` — النسخة الأفقية.
- `logo-animated.json` — نسخة Lottie المتحركة، إن وجدت.
- `app-icon-1024.png` — أيقونة التطبيق.

يفضل الاحتفاظ بنسخ RTL/English إن اختلف ترتيب العناصر. التطبيق يستخدم حاليًا علامة مؤقتة إلى أن تصل النسخة النهائية.
