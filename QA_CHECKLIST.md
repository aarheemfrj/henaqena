# هنا قنا — قائمة تحقق الإصدار الحالي

## تم التحقق منه

- Flutter: `flutter analyze` بدون أخطاء.
- Flutter: `flutter test` ناجح.
- API: `npm run build` ناجح.
- لوحة الإدارة: `node --check app.js` ناجح.
- `/health` يرجع حالة التشغيل.
- endpoints الإدارة مرفوضة بدون `x-admin-key` وتعمل بالمفتاح المحلي.
- مقدمو الخدمات التجريبيون يأتون من PostgreSQL المعزولة على المنفذ `5433`.

## قبل الإطلاق العام

- استبدال مفتاح الإدارة المحلي بمتغير سري في بيئة الاستضافة.
- توصيل مزود واتساب/SMS والبريد الفعلي.
- إضافة اختبارات تكامل لتسجيل الحساب والمراجعة والإعلانات.
- اختبار الأجهزة الحقيقية، الصلاحيات، الخرائط، والإشعارات.
- مراجعة سياسات الخصوصية وشروط الاستخدام قبل النشر.
